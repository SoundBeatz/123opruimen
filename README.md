# 123 Opruimen

Premium, responsive website voor **123opruimen.nl**.

## Website

De site is volledig statisch en bestaat uit:

- `index.html`
- `styles.css`
- `script.js`
- `logo.svg`

## Publiceren

De website kan rechtstreeks als statische site worden gepubliceerd via GitHub Pages, Hostinger of een andere webhost.

### Hostinger

Upload de bestanden naar de document root van het domein, meestal `public_html`, of koppel deze GitHub-repository aan de hosting als automatische deployments beschikbaar zijn.

### GitHub Pages

Open **Settings → Pages**, kies **Deploy from a branch**, selecteer `main` en `/root`.
